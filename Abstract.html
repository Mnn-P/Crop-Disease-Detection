Abstract

Crop disease detection is essential for improving agricultural productivity and reducing pesticide use. This project focuses on using CNNs and Federated Learning (FL) to build efficient models for detecting diseases from images of crops. The use of FL ensures data privacy and security, making it an ideal solution for decentralized agricultural applications.
