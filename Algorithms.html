Algorithms Used for Crop Disease Detection

The following algorithms were used in this project for detecting diseases in crops using Convolutional Neural Networks (CNNs) and Federated Learning:

  • CNNs (Convolutional Neural Networks): Used for feature extraction and image classification.
  • Federated Learning: A decentralized method allowing training across multiple devices without sharing private data.
  • ResNet50: A powerful CNN architecture used for image classification tasks in this project.
