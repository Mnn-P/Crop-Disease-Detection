Algorithms

The following algorithms were used in this project:

  • CNNs (Convolutional Neural Networks): Used for feature extraction from images.
  • Federated Learning: A decentralized approach where each client trains a local model.
  • ResNet50: A deep CNN model used for image classification, showing the best performance in this project.
